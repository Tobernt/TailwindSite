React Demo

Design system primitives

This page documents the shared React components that live in /design-system and ship with the Vite app in /react-demo. Each example shows usage, accessible patterns, and recommended props.

How to import

Use named imports from the package root so components stay consistent between projects: 

import { Button, Badge, Card, Input, Modal, Accordion } from '../../design-system';

The components are written in JSX and styled with the shared Tailwind preset.

Accessibility highlights

  • Buttons and links include visible focus outlines and disabled states.
  • Inputs wire labels to form controls with aria-invalid and helper text IDs.
  • Modals trap focus visually, close on Esc, and use role="dialog" plus descriptive IDs.
  • Accordions expose aria-expanded and aria-controls on toggles.
